## Usage Notes

- **Total: 40 image files** (13 tiles + 4 backgrounds + 6 frames + 10 UI + 4 icons + 4 summer)
- Replace the `.svg` placeholder files with `.png` (or `.jpg` for backgrounds)
- The `AssetLoader.js` will automatically try `.svg` fallback if `.png` is not found
- Update the file extensions in `js/config.js` ASSETS manifest if using different formats
- Estimated total size: ~2-3 MB (tiles ~15-30KB each, backgrounds ~100-200KB, UI ~5-20KB each)
